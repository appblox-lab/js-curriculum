## **Switch Statement**

`switch statement` is used to execute one code from multiple expressions. It is just like else if statement.

Syntax
  ```js
    switch(expression){  
        case value1:
            code to be executed;
            break;
        case value2:
            code to be executed;
            break;
            ......  
        default:   
            code to be executed if above values are not matched;  
    }    
  ```
